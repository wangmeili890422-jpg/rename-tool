# 🌿 图片批量重命名工具

上传 Excel + 图片文件夹，一键按商品名称批量重命名。

> 全程浏览器本地运行，无需安装任何软件，数据不上传服务器。

---

## 📖 如何使用

### 第一步：获取工具

点击本页面的 `rename-tool.html` → 右上角点 **「下载」** 按钮，保存到本地。

### 第二步：双击打开

直接 **双击** `rename-tool.html`，浏览器会自动打开工具界面。

### 第三步：操作流程

1. **上传 Excel 文件** — 第1列为商品名称，从第2行开始（可自定义列号和起始行）
2. **上传图片文件** — 选择多张图片，工具按文件名排序后依次对应 Excel 中的名称
3. **预览 & 下载** — 可选加前缀（如"列表图-"、"封面-"、"详情-"），确认后一键下载

> 超过3张图片自动打包为 ZIP 下载。

---

## 📋 Excel 格式要求

| 列号 | 内容 | 说明 |
|------|------|------|
| 第1列 | 商品名称 | 默认第1列，可在工具中修改 |
| 第1行 | 表头 | 默认跳过，从第2行开始读取 |

**示例：**

| 商品名称 | 价格 | 库存 |
|----------|------|------|
| 体检套餐A | 299 | 50 |
| 体检套餐B | 599 | 30 |
| 体检套餐C | 899 | 20 |

## 🛠 技术说明

- 纯前端 HTML 文件，即开即用
- 使用 [SheetJS](https://sheetjs.com/) 解析 Excel
- 使用 [JSZip](https://stuk.github.io/jszip/) 打包 ZIP 下载
- 支持 JPG / PNG / WebP 等常见图片格式
- 支持拖拽上传
